# Academic Information System Whitebox and Blackbox Testing
### Academic Information System Consist of 16 form as follows:
- Login
- Change Password
- Forgot Password
- Search
- Form Siswa
  * View Jadwal
  * View Nilai
  * Input & Print Surat
- Form Guru
  * Update & Input Nilai
  * Jadwal
- Form Staff
  * Input Data Guru
  * Input Data Staff
  * Input Data Siswa
  * Input Data Siswa Baru
  * Input Data Kepala Sekolah
  * Input Mata Pelajaran
  * Input Jadwal
  * View Data
  * Create ID 
  * Edit Data
- Form Kepala Sekolah
  * View Data Guru
  * View Data Siswa
  * View Data Staff
  * View Mata Pelajaran
  * View Jadwal
  * Confirm Surat Mutasi
 
### Whitebox testing will be test on the source code as follows:
- Login
- Change Password
- Forgot Password
- Search

### Blackbox testing will be test on 16 form on the system.
